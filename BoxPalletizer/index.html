Ключевые возможности

Оптимальная укладка ящиков

Программа подбирает оптимальную схему укладки в слое с максимальным количеством ящиков при соблюдении ограничений по габаритам, свесам и устойчивости.

Два варианта укладки:

  • Стандартная — простая схема с минимизацией зазоров у борта ящика.
  • Спиральная — специфическая укладка, повышающая устойчивость и равномерность нагрузки.

Если оба варианта дают одинаковое количество ящиков в слое, можно заранее задать предпочтение: стандартная или спиральная.

Учитывается кантование: укладка на дно, на ребро и на торец.

Расчёт нагрузки и прочности

Расчёт выполняется по ГОСТ 9142-2014 для тары из гофрокартона.

Выводятся ключевые параметры (в зависимости от включённых опций):

  • нагрузка на нижний ящик (кг/Н);
  • допустимая нагрузка при штабелировании и запас прочности;
  • сопротивление торцевому сжатию (ECT);
  • тип/марка гофрокартона и влияние ориентации гофры.

3D-визуализация

Понятная схема для работников склада. Четные и нечетные слои укладки отобрадаются разным цветом. Возможен поворот наждого слоя на 180 градусов относительно соседних слоев для более прочной укладки (связка слоев). На схему можно добавить текстовый блок с параметрами оптимальной укладки. Схему можно скопировать в буфер обмена и вставить в любой офисный документ (Word, Excel, PowerPoint)

Отчёт Word с шаблоном пользователя

Программа формирует подробный отчёт в формате Microsoft Word, который может быть настроен под шаблон пользователя.

Вы можете использовать собственный бланк предприятия: логотип, реквизиты, фирменные шрифты, структуру документа — отчёт автоматически заполняется рассчитанными данными.

Это удобно для передачи схем укладки в производство, логистику или заказчику без ручного оформления.

Key features

Optimal stacking logic

The software calculates the most efficient box layout per layer, maximizing box count while respecting stability and overhang limits.

Two stacking modes:

  • Standard — a simple layout that minimizes gaps at the box edge.
  • Spiral — specific layout that increases stability and load uniformity.

If both modes produce the same box count per layer, the user can define which mode has priority.

Box orientation can be considered: upright, on edge, or on side.

Load and strength calculation

Load calculations are performed in accordance with GOST 9142-2014 for corrugated cardboard packaging.

Key output parameters (depending on enabled options):

  • load on lower boxes (kg/N);
  • allowable stacking load and safety margin;
  • edge crush resistance (ECT);
  • cardboard type and flute orientation influence.

3D visualization

A clear diagram for warehouse workers. Even and odd stacking layers are displayed in different colors. Each layer can be rotated 180 degrees relative to adjacent layers for more secure stacking (layer linking). A text block with optimal stacking parameters can be added to the diagram. The diagram can be copied to the clipboard and pasted into any office document (Word, Excel, PowerPoint).

Word & PDF reports

Detailed reports can be generated in Word and PDF formats. Word reports support custom user templates (company branding, layout), automatically filled with calculated data.

Flexible parameters

Any box/pallet sizes, height/weight limits, overhang settings and non-standard packaging.

Palletizer PRO: watch mode

The PRO version can monitor a selected directory, automatically run the calculation when a JSON file with palletizing parameters appears, and save the PDF/JPG report to the selected output directory.

After successful processing, the source JSON file can be deleted or moved to an archive directory.

Available languages: English, Russian, Hebrew, German and Italian.
