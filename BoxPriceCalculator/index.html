Box Price Calculator — программа для расчёта стоимости изделий из гофрокартона: гофроящиков, гофролистов и гофрорулонов. Учитывает материалы, печать, упаковку и доставку, формирует техкарту и отчёт в PDF.

Что вы получаете

✔ единый расчёт для коробов, листов и рулонов

✔ прозрачные параметры: обрезь/отход, площадь заготовки, потребность картона

✔ учёт печати, упаковки и доставки в себестоимости

Программа работает в Windows и не требует установки базы данных — параметры и прайс хранятся в файлах.

Как это выглядит в работе

Главное окно: ввод параметров, расчет стоимости, предупреждения по ограничениям и построение развертки/чертежа.

Как это работает

1) Выбор изделия

Вы выбираете тип изделия: гофроящик (Д×Ш×В), гофролист (Д×Ш) или гофрорулон (тип и длина намотки), задаёте размер партии и ключевые параметры.

2) Параметры и цены

Выбираете профиль (B/C), марку картона (T-21, T-22 и т. п.) и цены. Программа может подставлять сохранённые значения из прайс-файла.

3) Итог и документы

Получаете себестоимость и стоимость партии, детализацию по статьям, а также PDF-техкарту и PDF-отчёт для производства/продаж.

Встроенный паллетировщик пачек

В состав программы входит паллетировщик пачек готовой продукции, позволяющий рассчитать оптимальную укладку пачек на поддон после упаковки гофроизделий.

Паллетировщик учитывает размеры пачки, размеры поддона, допустимые свесы, количество слоёв и ориентацию пачек. Это позволяет заранее определить:

  • сколько пачек помещается на один поддон;
  • высоту и устойчивость укладки;
  • количество поддонов на заказ;
  • эффективность использования объёма.

Расчёт особенно полезен при планировании отгрузок, хранении на складе и согласовании логистики с заказчиком.

Скриншот показывает расчёт укладки пачек на поддон с визуализацией слоёв.

Ключевые возможности

Короба / листы / рулоны

Единый инструмент для трёх типов изделий — удобно, когда в заказах встречаются разные форматы.

Обрезь/отход, площадь заготовки, количество заготовок на листе, потребность картона (м²/кг) и другие показатели, которые нужны технологу и экономисту.

Техкарта и отчёт (PDF)

Быстро формируете документы для производства и для клиента — единый формат и единый стиль.

Развертка и схемы

Программа может строить развертку гофроящика и выдавать вспомогательные схемы, полезные в работе технолога.

Файлы данных без базы

Настройки и справочники хранятся в файлах: их легко копировать, переносить, резервировать и версионировать.

Программа может проверять наличие обновлений при запуске (проверку можно включать/отключать). Отдельно можно обновлять установленные словари/языковые модули.

Практично: пользователи получают улучшения и обновлённые словари без “ручной установки”.

Автоматизация “дежурством”

Программу можно поставить на “дежурство”: она следит за указанной папкой и, если появляется JSON-файл с данными расчёта, выполняет расчёт автоматически и сохраняет техкарту и отчёт в заданные каталоги.

Это удобно для интеграции с сайтом/CRM: заказ → JSON → автоматический расчёт → готовые PDF-документы.

How it looks in real work

Main window: input parameters, cost calculation, constraint warnings, and blank layout/drawing preview.

How it works

1) Choose product type

Select: corrugated box (L×W×H), sheet (L×W) or roll (roll type + winding length), then set batch size and key parameters.

2) Enter parameters & prices

Choose flute profile (B/C), board grade (T-21, T-22, etc.) and pricing. The program can auto-fill saved values from a price list file.

3) Results & documents

Get unit cost and batch cost, detailed breakdown by cost items, plus a PDF tech card and PDF report for production and sales.

Built-in bundle palletizer

The software includes a built-in bundle palletizer designed to calculate optimal stacking of finished product bundles on pallets after packaging.

The palletizer takes into account bundle dimensions, pallet size, allowed overhangs, number of layers and bundle orientation. This allows you to determine in advance:

  • how many bundles fit on a pallet;
  • overall pallet height and stability;
  • required number of pallets per order;
  • storage and transport space efficiency.

This calculation is especially useful for shipment planning, warehouse storage and logistics coordination with customers.
